Recap completo del codice

Questo codice simula la camminata di un quadrupede controllando 8 servomotori (2 per ogni zampa) utilizzando animazioni sinusoidali. La simulazione include controlli interattivi per avviare, fermare e direzionare il movimento (avanti, indietro, destra, sinistra) e consente di regolare ampiezza e fase di ogni servomotore tramite slider. Un contatore di frame mostra il frame corrente.

for i, (line, point) in enumerate(zip(lines, points)):

amplitude = amp_sliders[i].val

phase = phase_sliders[i].val + frame * 0.1 * direction + (i % 2) * lateral_shift

y_data = amplitude * np.sin(x + phase)

line.set_ydata(y_data)

point.set_xdata(x[frame % 1000])

point.set_ydata(y_data[frame % 1000])

  • update_movement: Aggiorna il movimento delle sinusoidi e dei punti.
  • Il parametro direction controlla se il movimento avanti (1) o indietro (-1).
  • Il parametro lateral_shift controlla i movimenti laterali.

Ecco una spiegazione dettagliata della funzione update_movement:

Funzione update_movement(frame, direction=1, lateral_shift=0)

Questa funzione aggiorna il movimento di tutti gli 8 servomotori in base al frame corrente dell'animazione. Ogni servomotore segue una traiettoria sinusoidale che pu essere modificata in ampiezza, fase, direzione e spostamento laterale.

Parametri della funzione

  • frame: Il numero del frame corrente dell'animazione. Viene utilizzato per calcolare la posizione attuale del movimento.
  • direction: Indica la direzione del movimento. Pu essere 1 (avanti), -1 (indietro) o altri valori per regolare la velocit del movimento.
  • lateral_shift: Indica uno sfasamento laterale della fase per ottenere movimenti destra/sinistra.

Descrizione del ciclo

python

Copy code

for i, (line, point) in enumerate(zip(lines, points)):

  • i: L'indice del servomotore (da 0 a 7, poich ci sono 8 servomotori).
  • line: La linea sinusoidale associata a questo servomotore.
  • point: Il punto rosso mobile lungo la traiettoria della sinusoide.

Calcolo dei parametri

python

Copy code

amplitude = amp_sliders[i].val # Ottiene il valore di ampiezza dal cursore per questo servomotore

phase = phase_sliders[i].val + frame * 0.1 * direction + (i % 2) * lateral_shift # Calcolo della fase per il servomotore

offset = (i // 2) * np.pi / 4 # Offset per sfasare le zampe di ogni coppia (anteriore/posteriore)

  1. amplitude: La posizione verticale della sinusoide moltiplicata per il valore di ampiezza fornito dallo slider.
  2. phase: La fase della sinusoide calcolata come la fase di base (phase_sliders[i].val) pi una variazione proporzionale al frame. L'aggiunta (i % 2) * lateral_shift garantisce uno sfasamento laterale specifico per il movimento destra/sinistra.
  3. offset: Ogni coppia di zampe (due per ogni riga) ha uno sfasamento costante per garantire che le zampe anteriori e posteriori si muovano in modo coordinato ma opposto.

Calcolo della sinusoide

python

Copy code

y_data = amplitude * np.sin(x + phase + offset) # Calcola la sinusoide aggiornata

line.set_ydata(y_data) # Aggiorna i dati della sinusoide nel grafico

La nuova traiettoria della sinusoide viene calcolata con la nuova fase e ampiezza. Poi viene aggiornata con set_ydata(y_data) per farla apparire visivamente aggiornata nel grafico.

Aggiornamento del punto mobile

python

Copy code

t = frame % 1000 # Determina la posizione attuale del punto sulla sinusoide

point.set_xdata(x[t]) # Aggiorna la posizione orizzontale del punto

point.set_ydata(y_data[t]) # Aggiorna la posizione verticale del punto

  • t: Determina la posizione corrente del punto mobile sulla sinusoide. Usa il modulo % 1000 per assicurarsi che il punto rimanga all'interno dei limiti della lunghezza della sinusoide (x ha 1000 punti).
  • set_xdata e set_ydata: Aggiorna la posizione del punto rosso sulla sinusoide in modo che segua la traiettoria.
